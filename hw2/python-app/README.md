# Part 2 and 3: Connecting the Database to a Traditional App and Building an MCP Server

This folder contains files for Parts 2 and 3 of Homework 2, where you will connect your database functions to a traditional Python application and then build an MCP server to provide board game recommendations.

## Project Structure

```
.
|-- test_data/              # Test data for MCP server tests
|-- .gitignore              # Git ignore file
|-- app.py                  # Traditional Python TUI application connected to the database
|-- test_db_app.py          # Tests for the database functions used in app.py
|-- mcp_server.py           # MCP JSON-RPC server implementation
|-- db.py                   # Database Manager File (you will modify this)
|-- agent_client_ollama.py  # Interactive Ollama-based client
|-- test_mcp_server.py      # MCP tests
|-- requirements.txt        # Python dependencies
|-- README.md               # This file
```
